## Installation
1. Clone the repository or download the project.
git clone https://github.com/AteetVatan/WikiSaver.git

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   pip install  openai
   pip install google-generativeai
   ```
3. Run the project:
   ```bash
   python3 main.py
   ```
